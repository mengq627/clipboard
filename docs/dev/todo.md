- [x] 分组列表刷新：当内容分组时，仅分组的内容数量刷新，而不是整个分组的列表刷新
- [x] 当应用开启，但是失去焦点时，win+v重新聚焦，而不是隐藏应用
- [x] 监听系统剪切板从定时轮询修改为事件监听回调
